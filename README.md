# Polynomial Series Analyzer

A Python tool to automatically analyze numerical series and generate:

- The **general nth term equation**
- The **closed-form equation for the sum of the series**

Unlike traditional arithmetic progression solvers, this project supports **series generated by polynomial functions of any degree** (linear, quadratic, cubic, etc.).

The algorithm detects the degree of the generating polynomial using the **finite difference method** and constructs the corresponding symbolic expression.

## How It Works

The algorithm uses the **finite difference method**.

Example:

Series

```
1 4 9 16 25
```

First differences

```
3 5 7 9
```

Second differences

```
2 2 2
```

Since the **second difference is constant**, the series is generated by a **quadratic polynomial**.

The program reconstructs the polynomial expression for the nth term.

## Usage

Example:

```python
from nth_term_of_series import nth_term_of_series
from sum_of_series import sum_of_series

series = [1, 4, 9, 16, 25]

nth_term = nth_term_of_series(series)
sum_formula = sum_of_series(series)

print(nth_term)
print(sum_formula)
```

Output

```
General equation of nth term of series is: n^2
General equation of sum of series is: n(n+1)(2n+1)/6
```
